🚗 Vehicle Rental System (Java):

A simple console-based Vehicle Rental System built using Java.
This project demonstrates Object-Oriented Programming (OOP) concepts like inheritance, abstraction, and polymorphism.

Features:
Customer information input
Multiple vehicle types:
Car 🚗
Bike 🏍️ (10% discount)
Truck 🚚 (+20% extra charge)
Rental cost calculation based on days
Clean console-based UI

Technologies Used:
Java
OOP (Inheritance, Abstraction, Polymorphism)

How to Run:
Compile the program:
javac VehicleRentalSystem.java
Run the program:
java VehicleRentalSystem

Project Structure:
VehicleRentalSystem.java

OOP Concepts Used:
Abstract Class → Vehicle
Inheritance → Car, Bike, Truck
Encapsulation → Customer class
Polymorphism → calculateCost()
